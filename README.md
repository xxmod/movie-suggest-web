# 电影电视剧愿望单网站

> 使用AI生成的愿望单功能，使用TMDBapi获取电影电视剧数据，快捷的愿望单网页

## 快速启动

1.clone这个仓库，解压

2.修改.env.sample为.env,打开.env修改其中tmdbapi、端口、管理密码以及邮箱相关设置

3.打开cmd，执行 `npm install`与 `npm run start`（自行下载node）


## 功能

* 透过IMDB搜索相应电影电视剧，添加愿望单
* 正确配置邮箱通知可以在有人添加愿望单后邮箱自动通知管理员
* 后台可以看到添加的愿望单，可以给予反馈
